# Desabilitar Atributo `required` em todos os campos do Formulário

Este repositório contém um script simples para percorrer todos os inputs de um formulário HTML e remover o atributo `required`. Essa funcionalidade pode ser útil durante o desenvolvimento front-end, facilitando a implementação e teste de validações dos formulários.

## Como Usar

Abra o arquivo index.js
Copie a função removeRequiredFields e cole em seu script (ou no console do navegador)
Chame a função passando o ID do seu formulário como parâmetro
